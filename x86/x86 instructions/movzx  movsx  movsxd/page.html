move with zero extention / with sign extention
works same as mov, no memory to memory
used when moving small values into bigger registers

movzx - move, zero extend (for unsigned), works with 8,16,32 bit

movsx - move, sign extend, deals only with 8 and 16 bit

movsxd - move, sign extend, dword, deals with 32 bit values sign extention (signed dword extend)
