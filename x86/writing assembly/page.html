2 ways for writing assembly:
  • Inline Assembly - a way to include assembly inline with C/++ code.
  • Standalone Assembly - pure assembly written to a file (not inside the main C/++ file), assembled into an object file and then linked into the final binary with other compuled objects.

Note that different assemblers use different syntaxs, most common being MASM in MSVS and NASM.
Raw bytes can also be written in order to avoid finding the right syntax.

Inilne assembly with GCC- uses GNU Assembler (gas) syntax (== AT&T). in the form of:
asm("instructions separated by \n"); // using __asm__ is also valid function name in case of name conflicts
