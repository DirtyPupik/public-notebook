The unary operator * (Asterisk) is the dereferencing operator.

If x is a pointer, then *x is what x points to.
The unary operator & (Ampersand) is the address-of operator.
If x is anything, then &x is the address at which x is stored in memory.
The * and & operators are inverses of each other: if x is any data, and y is any pointer, then these equations are always true:  
*(&x) == x
&(*y) == y
