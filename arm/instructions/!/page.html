! (exclamation mark) means register write-back.
changes are done to the base register

str r2, [r1, #4]! - Store the value found in R2 to the memory address found in R1 plus 4.
              • Base register (R1) is modified prior to operation : R1 = R1+4.
              • this is called pre-indexed addressing mode.
note that base register may also be modifed after the opertaion:
ldr r3, [r1], #4
