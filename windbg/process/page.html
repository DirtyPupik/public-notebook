Documentation for (Display information) 
!process
Documentation for (Set Process Context) 
.process

When examining the current process context in windbg, DirBase points to the directory base address of the process (i.e PML5\4 \ PDPT..).
It has the same value of the CR3.


Display the current process context
  • !process -1 [flags]
  • 0: kd> !process -1 0
Listing all processes:
  • !process 0 [flags]
  • 2: kd> !process 0 0

Search for the process context based on executable name:
  • !process 0 [flags] [exe name]
  • 0: kd> !process 0 0 notepad.exe

Search for the process context based on executable process ID (PID):
  • !process [PID] [flags]
Where the Process ID can be found with tools like Task Manager.
  • 0: kd> !process 0x1b44 0

Changeprocess context to target executable:
  • .process /i /r /p ["PROCESS" address from !process output]
  • 0: kd> .process /i /r /p ffffcd089d5d4300
You need to continue execution (press 'g' <enter>) for the context
to be switched. When the debugger breaks in again, you will be in
the new process context.
