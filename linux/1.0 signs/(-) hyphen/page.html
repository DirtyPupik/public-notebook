(-) Hyphen
Stands for stdin or stdout, case dependent.
Some commands accept - in place of a filename.

Example:
For the line of code:
wget -O - "https://www.dropbox.com/download?plat=lnx.x86_64" | tar xzf -  

‘
-O file
’
  is for writing to a file.
in this case the file is - (standard output).
whatever's written to stdout, is piped to stdin of tar.
tar extract the whatever is coming from stdin.
