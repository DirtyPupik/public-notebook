printenv - print all enviroment variables

the common use is lowercase for local and UPPERCASE FOR GLOBAL

new env vars are not permament!

print a specific one:
printenv HOME
echo $HOME

to create a local variable- use lowercase
myname="pupik"
echo $myname

to create global variable - use UPPERCASE
export myname - will export the local variable we created to global env

to remove the variable
unset myname

to set permament:
subl ~/.profile
add the "export <varname>='a string'" at the end of the file
. ~/.profile
