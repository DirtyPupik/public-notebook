find . -name "filename*" search in current dir and all subdirs

  • / (slash) — search the whole system.
  • . (dot) — search from the folder you’re currently working on (current directory).
  • ~ (tilde) — to search from your home folder.


-iname (case) instsensitive
find . -iname my-file


-not does not contain name
find . -not -name my-file


search by format (*)
find . -name “*.txt”


find and delete
find . -name my-file -delete


search by type
  • d – directory or folder
  • f – normal file
  • l – symbolic link
  • c – character devices
  • b – block devices
example:
find / -type f -name my-file
