dmesg prints the kernel ring buffer logs

kernel keeps all the logs in a ring buffer
this is done to avoid boot logs beeing lost until the syslog daemon starts and collects them and store them in var/log/dmesg

dmesg -c will clear ring buffer after printing kernel
dmesg -C will clear ring buffer, WILL NOT print kernel
dmesg -t will NOT print timestamp
dmesg -T will convert timestamp into human readble format
