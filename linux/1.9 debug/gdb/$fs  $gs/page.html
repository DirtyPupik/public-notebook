To access fs and gs register during runtime
$fs_base
$gs_base

Example - for the instruction:
mov %fs:0x28,%rax

One can print the register base address:
(gdb) x/gx $fs_base
0x7ffff7fb7580: 0x00007ffff7fb7580

To access the location in memory, add the offset to the base register:
(gdb) x/gx 0x7ffff7fb7580+0x28
0x7ffff7fb75a8: 0x58f6b1f9673d2400
