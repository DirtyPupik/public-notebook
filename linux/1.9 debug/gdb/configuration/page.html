It is possible to create a GDB configuration for commonly used commands.
Commands may be saved inside a plain text file.
The file may be executed on startup with -x command

Example 1:
display/10i $rip
display/10xg $rsp
display/x $rax
start

Example 2:
display/10i $rip
display/x $rbp
display/x $rsp
display/x $rax
display/x $rbx
display/x $rcx
display/x $rdx
display/x $rdi
display/x $rsi
display/x $r8
display/x $r9
display/10gx $rsp
start
