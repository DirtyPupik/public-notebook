Print- can print a single value/register (may be used as "p"):
  • Size letters are meaningless in "print" command.
  • Size is addressed by the register, via intel syntax:
    • (gdb) p/x $r14
    • $1 = 0xdeadbabe
    • (gdb) p/x $r14w
    • $2 = 0xbabe
  • Item count other than 1 (number) is meaningless in "print" command.
  • p/x $ah
