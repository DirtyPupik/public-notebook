Registers- can be accessed as such:
$sp - stack pointer
$pc - program counter
$fp - frame stack pointer
$ps - processor status


Changing Registers:
  • set $sp += 4
  • set $r14 = 0xdeadbabe
Example:
(gdb) set $rax = 0xdeadbeeff00dface
(gdb) p/x $rax
$15 = 0xdeadbeeff00dface
(gdb) set $ax = 0xcafef00d - NOTE: ax is 2 byte, therefore it will take only the first 2 bytes
(gdb) p/x $rax
$16 = 0xdeadbeeff00df00d - "face" changed to "f00d" as we changed only ax.


Viewing registers:
info r rax rbx rsp - will give info about the 3 registers.
