opening a file:
from terminal, gdb ./filename or gdb <filename>
from inside gdb, file command may be used
-q quite
-fno-stack-protector
-static do not use dynamic linked library


running:
run / r <arg1> <arg2> ...
start start and break at entry point.
c - continue
s - step


Disassembling
disas disas at current location
disas <address or symbol name>
disas /r display raw byte opcodes


Continuously-updating Disassembly Display
display/10i $rsp


Watchpoints - are utilizing hardware breakpoints behind the scene
watch <address> Break On Write-Only Access
rwatch <address> Break On Read-Only Access
awatch <address> Break On Read-And-Write Access

Setting the access-size of watchpoints - can be done with C type casting
  • watch *(char *) <address> will set break if 1 byte is accessed from address (address+2 wouldnt set break)
  • watch *(long long *) <address> will set break if any of 8 bytes starting from address are acceessed.
