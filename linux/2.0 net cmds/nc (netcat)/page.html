TRANSPORT LAYER CONNECTIVITY TOOL

linux - netcat - run by "nc" syntax: nc <host> <port>
it establishes TCP connection with the host, data can be sent
flags:
-z zero input/output mode
-v verbose - make the output understandable to humans (non-verbose output is good for use in scripts, verbose literal meaning is "talking too much")
-vv very verbose - more output text
-u start UDP instead of TCP

to restart the service:
systemctl restart ssh.service
