a server that "acts on behalf"
can be front proxy to serve the client
or reverse proxy to serve a server

proxy can provide: anonymity, security, content filtering, increased prefomance and more
a gateway router meets the definition of a proxy
proxies exist in every layer, and vary in type

web proxy - back than one of the uses was to catch conten, but nowdays connection is fast enough for this to be redundent, also web has become very dynamic
web proxy can act as a firewall and forbid access to some sites (lets say no twitter on school network)

reverse proxy - appear as single server to many clients but represents many servers behind it, it can distribute request, similar to the DNS round robin load balancing

proxies also help encryption (?)
