ARP the relating protocol between the data link layer and the network layer

ARP is a protocol used to discover the hardware address of a node with a certain IP address.

Almost all network connected devices will retain a local ARP table. An ARP table is just a list of IP addresses an the Mac addresses associated with them.
(the values of IP to MAC are assigned by DHCP)

if there is no data of an IP in the ARP table, the sending device sends a broadcast ARP msg over the broadcast address (255.255.255.255 or FF:FF:FF:FF:FF:FF)

the recieving end sends back an ARP responsecontaining the MAC address assosiated

the MAC\IP pair will be stored in a local ARP table with short expiry time
