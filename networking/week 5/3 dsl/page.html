digital subscriber line

using a different frequency to send data over phone lines, the different freq ensures that the connection will not interfere with the rest of the data passing on the line

DSL uses DSLAMs DSL Access Multiplexers

unlike dial up connection, DSL is long running - the connection is established till the DSLAM is powerd off

ADSL - Asymetric DSL - feature different speed for download and upload

SDSL - symmetric

most DSL tech have rate limit of 1.544 (same as T1)

HDSL - High bit rate DSL - provides rates above 1.544
