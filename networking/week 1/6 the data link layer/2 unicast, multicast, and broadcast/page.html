On the network 192.168.0.0/24 the usual gateway is 192.168.0.1 and the network broadcast address is 192.168.0.255.

/24 is the boradcast range (24 first bits eg 255.255.255.0)

The unicast address will have the value of the MAC address of the destination device.

Multicast frames have a value of 1 in the least-significant bit of the first octet of the destination address.
This helps a network switch to distinguish between unicast and multicast addresses.
examples of an Ethernet multicast address
01:00:0C:CC:CC:CC (00000001 : 00000000 : 00001100 : 11001100 : 11001100 : 11001100)
0100.CCCC.DDDD (0:1 = 0000:0001)
0900.AAAA.BBBB (0:9 = 0000:1001)

broadcast is 255.255.255.255 (all one's in binary)
