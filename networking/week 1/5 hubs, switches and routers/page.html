Hubs and Switches - connects devices in a single network (LAN)

hubs - layer 1 device - all devices talk to each other - broadcast all day long. this creates collison domain

collision domain, is a network segment where only one device can communicate at a time.

switch - layer 2 device - connects only 2 computers in communication (no broadcast ADL)

Router - layer 3 device - connects individual nodes
Routers share data with each other via a protocol known as BGP, or border gateway protocol, that let's them learn about the most optimal paths to forward traffic.



all of the above are considered to be NODES
